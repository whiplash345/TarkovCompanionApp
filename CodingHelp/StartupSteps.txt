Startup:

1. open powershell and cd to the 'myapp' folder
2. run 'git pull' if changes were made on other machine
3. run 'npm ci' if dependencies were changed on other machine
4. Start Android Emulator
5. run 'adb devices' to confirm emulator running
6. run 'adb reverse tcp:8081 tcp:8081' to prevent emulator from loosing connection
7. run 'npx expo start -c' to start expo
8. press 'a' to start running in android emulator


Shutdown:
1. stop expo 'Ctrl+C'
2. run 'git add .' in the VSCode Terminal
3. run 'git commit -m "my message" in the VSCode Terminal
4. run 'git push' in the VSCode Terminal
